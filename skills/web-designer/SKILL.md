---
name: web-designer
description: Design and critique website interfaces, landing pages, and content layouts. Use when an agent needs to improve visual hierarchy, conversion flow, responsiveness, trust signals, readability, or produce design direction for web pages.
---

Use this skill for practical web design work.

Core tasks:

1. Evaluate layout, hierarchy, spacing, typography, and visual consistency.
2. Improve calls to action, trust elements, and conversion-oriented page flow.
3. Review mobile responsiveness and content readability.
4. Turn vague design wishes into concrete page structure and UI recommendations.
5. Produce redesign guidance that developers and content editors can implement.

Workflow:

1. Define the page goal, audience, and desired user action.
2. Inspect above-the-fold clarity, navigation, CTA placement, and content rhythm.
3. Identify friction in hierarchy, scanning, trust, and mobile behavior.
4. Propose a stronger visual and structural direction with concrete changes.
5. Present recommendations in implementation-ready language.

Deliverables should include:

- design diagnosis
- UX and conversion issues
- layout and hierarchy recommendations
- mobile and accessibility notes
- suggested section order or page wireframe outline
