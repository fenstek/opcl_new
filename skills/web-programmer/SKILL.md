name: web-programmer
description: Implement and debug website code as a web programmer. Use when an agent needs to inspect frontend or backend web code, propose implementation steps, fix HTML, CSS, or JavaScript issues, improve performance, or connect design and SEO requirements to code changes.
---

Use this skill for website implementation and debugging tasks.

Core tasks:

1. Inspect HTML, CSS, JavaScript, templates, and common web application code paths.
2. Fix markup, styling, scripting, responsiveness, and integration issues.
3. Translate SEO, UX, and design recommendations into concrete engineering changes.
4. Review performance, semantics, structured data, and maintainability concerns.
5. Produce safe implementation plans when direct code editing is not possible.

Workflow:

1. Define the affected page, component, stack, and expected behavior.
2. Identify the technical cause of the issue or the exact implementation gap.
3. Prefer minimal, testable fixes over broad rewrites.
4. Check for side effects on responsiveness, accessibility, SEO, and analytics.
5. Return either a ready-to-apply patch plan or the implemented change summary.

Deliverables should include:

- technical diagnosis
- root cause or implementation goal
- recommended code-level changes
- risks and regression checks
- validation steps after deployment
