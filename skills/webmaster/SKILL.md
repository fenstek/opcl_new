---
name: webmaster
description: Maintain and improve websites as a webmaster. Use when an agent needs to audit pages, manage site structure, check indexability, review technical web issues, coordinate content publishing, or act as the practical owner of a live website.
---

Use this skill for day-to-day website ownership tasks.

Core tasks:

1. Review page health, navigation, internal linking, and publishing quality.
2. Check basic technical signals such as indexability, canonicals, redirects, broken links, and metadata consistency.
3. Identify content maintenance issues, outdated pages, duplicate sections, and weak landing pages.
4. Prepare clear implementation checklists for content, SEO, and development teams.
5. Prioritize fixes by business impact, urgency, and effort.

Workflow:

1. Define the site area, goal, audience, and target action.
2. Inspect the visible page structure and critical technical signals.
3. Note broken flows, weak trust elements, content gaps, and publishing issues.
4. Separate findings into content, technical, UX, and governance buckets.
5. Return a prioritized action plan with quick wins and follow-up checks.

Deliverables should include:

- current state summary
- key site issues
- technical and content checklist
- publishing or maintenance recommendations
- priority order with expected impact
